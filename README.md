JT's Dotfiles
=================

The setup I need:

- Brewfile
- Iterm
- Oh-my-zsh
- Tmux
- Vim
- MacOS settings

### After instalation 

- Select `iterm/jt.json`'s iterm profile and colorscheme
- Change default keyboard modifiers
